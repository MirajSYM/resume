Introduction

In the realm of application development, our team, led by yours truly, embarked on a transformative journey as part of the Application Development class. The result: Rent&go, a pioneering project aimed at streamlining car rentals through both web and mobile applications. Let's dive into the STAR method to understand the Situation, Task, Action, and Result of our endeavor.

Situation

Our mission was clear: create an intuitive and efficient car rental platform. Rent&go featured distinct modules, each designed to enhance the overall user experience.

Task

Modules Overview

  1. Login:
    • User registration with name, DOB, email, phone, and password.
    • Email verification for successful registration.
    • Secure login using email and password.
  2. Manage Driver (Admin):
    • Admin authority to add/remove drivers.
    • Storage of essential driver information.
  3. Booking:
    • Seamless vehicle booking via website or mobile app.
    • Admin control for booking approval and removal.
  4. Admin Update:
    • Admin empowerment to update/remove prices, bios, services, vehicles, and profiles.
  5. Cars:
    • Fleet management for admins, including vehicle addition, removal, and booking.
  6. FAQ:
    • Addressing common user questions for clarity.
  7. HelpDesk:
    • Direct user-admin communication through email.
  8. About:
    • Providing insights into the Rent&go platform.
  9. Payment:
    • Secure online transactions, including credit/debit card options.
  10. Navigation:
    • Assisting users and admins in locating vehicles.

Functional Requirements

Rent&go met a set of functional requirements ensuring a seamless user experience, from registration and password recovery to car browsing, booking, online payment, and communication.

Action

Utilizing PHP, HTML, CSS and MySQL for the web version, alongside Microsoft Visual Studio code, Flutter and IntelliJ for the mobile app, we crafted a dual-platform solution. Data modeling played a pivotal role in ensuring efficiency and scalability.

Result

Rent&go emerged as a resounding success, delivering a user-friendly platform for effortless car rentals. The amalgamation of web and mobile applications, coupled with adept technology choices, culminated in a robust and reliable system. Our commitment to meeting functional requirements yielded an application that redefined the car rental experience.
