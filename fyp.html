Save Me - Accident Alert & Notification System

  • Category: Embedded System
  • Tech Stack: NodeMCU, Flutter, Firebase
  • Project Demo: Click Here to Watch

The "Save Me" Accident Alert & Notification System aims to enhance emergency responses and reduce traffic fatalities. The project encompasses a mobile app that provides incident notifications and displays the location of the incident in real-time.

Scroll Down to View Project Details


Project Scope:

  • Enhancing emergency responses to accidents,
  • Providing incident notifications through a mobile app, and
  • Displaying the real-time location of incidents on the mobile app.

Project Design:

The system's significance lies in its potential to decrease traffic fatalities by shortening response times. The project, named "Alert and Notification," is designed to achieve this objective.

Use Cases:

  1. Sign Up: Describes the process when a new user attempts to use the application feature.
  2. Sign In: Follows successful registration to the application.
  3. Accident Location: Details how the user views the accident location on the application.
  4. Accident Notification: Describes the process of the user receiving accident notifications through the application.
  5. Alert: Focuses on the hardware aspect, explaining the alert that the driver receives if crossing the safety limit threshold.

The system's holistic approach, combining hardware alerts and mobile application features, ensures a comprehensive solution to enhance emergency responses and contribute to the overall reduction of traffic fatalities.
